# Projeto de Análise Automática de ORs

## Sobre o projeto
Este projeto foi desenvolvido em VBA para automatizar a análise de ORs (Ordens de Recebimento) dentro do Excel.

Ele reduz significativamente o tempo necessário para analisar grandes volumes de dados, utilizando os status sistêmicos disponíveis para classificar e organizar automaticamente as informações.

---

## Problema que resolve
Antes da automação:
- Análise manual de ORs levava cerca de **1 hora ou mais**

Depois da automação:
- Tempo reduzido para aproximadamente **15 a 20 minutos**

Além disso:
- Atualiza status automaticamente
- Evita erros manuais
- Padroniza a análise

---

## Tecnologias utilizadas
- Microsoft Excel
- VBA (Visual Basic for Applications)

---

## Como utilizar

1. Abra a planilha no Excel  
2. Pressione `Alt + F8`  
3. Selecione a macro criada  
4. Pressione `Enter`  

O processo será executado automaticamente.

---

## Observações
Este projeto foi criado com foco em resolver um problema real do dia a dia, otimizando tempo e aumentando a eficiência na análise de dados.
